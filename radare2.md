## Intro 

- i : Get info from opened file

## Trucs
- Faire un grep : utiliser ~, par exemple i~machine : fait un grep sur machine dans la sortie de la commande "i"
- Afficher l'aide : "?", exemple "i?" : affiche l'aide sur la commande "i"
